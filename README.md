# easyenv

一键初始化开发环境的桌面工具。基于 Tauri 2 构建。

## 功能

- 可视化管理开发环境，自定义创建配置
- 支持在线URL下载和本地文件安装
- 安装后自动检测 bin 目录并配置系统 PATH
- 支持配置安装后执行脚本（命令或脚本文件）
- 安装中支持终止并自动回滚
- 实时进度展示（操作描述 + 树形timeline）
- 安装日志自动保存到本地文件
- 数据持久化，重启不丢失
- 自定义窗口（无原生标题栏），系统托盘右键菜单
- 版本检查更新
- 11 种语言、白天/夜间主题

## 构建

```bash
git tag v0.2.0 && git push origin v0.2.0
```

## 本地开发

```bash
cargo install tauri-cli --version "^2"
cargo tauri dev
```
